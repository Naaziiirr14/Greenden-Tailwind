🌿 Greenden - Plant Products Website

Greenden is a simple and responsive plant products website built using HTML and TailwindCSS.
This project showcases different plant products with a clean UI and modern responsive design.

🚀 Features

🌱 Responsive design

🎨 Styled using TailwindCSS

🪴 Plant product showcase

📱 Mobile-friendly layout

⚡ Fast and lightweight

🧭 Simple navigation bar

🖼 Product cards with images and descriptions

🛠 Built With

HTML5

TailwindCSS
